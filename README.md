# 图片压缩
- png 走 pngquant --quality=65-80
- jpg / jpeg 走 MozJPEG

在

```shell
npm run png
```

```shell
npm run jpg
```

之前可以先安装对应工具(MacOS)

```shell
brew install mozjpeg pngquant
```
